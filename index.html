  • Instructions

    • Sentiment Analysis is the process of determining whether a piece of writing (tweets, reviews, etc.) is positive, negative or neutral.

    • For knowing the accurate result of the sentiments, one need to upload a comma-separated values (CSV) file format only, in the Uploads Section (03 menu option).

    • In CSV, each record consists of one or more fields which are separated by commas. Therefore, before uploading any csv files one should always check for the format of the labels carefully. It must be in this (,Sentiment,Tweet) format.

    • After the file is uploaded, one can view the classification report which consist of precision, recall, f1-score and support for the following values - Positive, Negative and Neutral.

    • One can click on Graphs Section(04 menu option) to view the Graph generated with different Twitter Sentiments.

    • Accuracy

      How often is the classifier correct? - Accuracy
      Accuracy = (True Positives + True Negatives) / (Total number of Predictions)

    • Precision, Recall and F1-Score

      When it predicts the positive result, how often is it correct? - Precision
      The number of positive identifications that actually belong to the positive class is known as precision. Precision = True Positive / ( True Positive + False Positive )

      When it is actually the positive result, how often does it predict correctly? - Recall
      The number of positive predictions that are identified correctly out of all positive examples. In other words, Recall is used when the goal is to limit the number of false negatives (FN).
      Precision = True Positive / ( True Positive + False Negative )

      It is the harmonic mean of precision and recall. - F1-Score
      F1-Score = 2 * ( precision * recall ) / ( precision + recall )

    • Support

      Support is the number of actual occurrences of the class in the particular dataset.
